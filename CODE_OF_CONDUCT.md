# Code of Conduct

## Standard

Be respectful, direct, and constructive.

Unacceptable behavior includes harassment, personal attacks, deliberate disruption, publishing private information, and any conduct that makes collaboration unsafe.

## Reporting

Report problems through GitHub issues or repository maintainers. Do not include sensitive local data, tokens, prompts, or private logs in public reports.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or contributions that violate this standard.
